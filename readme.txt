# Sales Management System

The main objective of the NodeJS Project On Sales Management System is to manage the details of  
Supplier, Customer,Inventory, Users Multiple companies, products perches, biling, Users . 

## Functionalities provided by NodeJS Project On Sales Management System are as follows:

- User Signup & User Login with JWT Authentication
- check Userâ€™s roles with record in database
- online offline status of frontend client
- validate input data
- Upload images
- old token will expire after getting new access token
- Two different environments (development and production)
- compare password with password in database

### Built with:

- [MongoDB](https://www.mongodb.com/) - The NoSQL database used
- [Socket.IO](https://socket.io/) - Online ofline status
- [bcryptjs](https://www.npmjs.com/package/bcryptjs) - Bcrypt password
- [jsonwebtoken](https://www.npmjs.com/package/jsonwebtoken) - To verify user
- [Mongoose](https://www.npmjs.com/package/mongoose) - Object Relational Mapper (ORM)
- [express-validator](https://express-validator.github.io/docs/) - validate any data coming in as input
- [Cloudinary](https://cloudinary.com/) - manage images
- [dotenv](https://www.npmjs.com/package/dotenv) - Environment variables


### development environment:

- npm run dev

### production environment:
